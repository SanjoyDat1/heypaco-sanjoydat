# Sanjoy Dat's portfolio

A personal portfolio built with **Next.js 15**, **Tailwind CSS v4**, and
handwritten, vendored UI primitives. Generated by [HeyPACO](https://heypaco.com).

> Want to change something? Chat with HeyPACO — we'll commit the diff
> back to this repo. Or edit any file yourself; the site is just
> React, no special framework.

## Running locally

```bash
npm install
npm run dev
```

Visit http://localhost:3000.

## Project layout

```
app/
  layout.tsx            # shared chrome (fonts, nav, footer)
  page.tsx              # home
  work/page.tsx         # all projects
  work/[slug]/page.tsx  # per-project case study
  about/page.tsx        # bio, skills, contact
components/
  sections/             # hero, projects, stats, ...
  ui/                   # Button, Card, Badge (vendored, no Radix runtime)
lib/
  site-data.ts          # all content — edit this to update the site
  utils.ts              # cn() helper
styles/
  globals.css           # Tailwind + design tokens
```

## Editing content

Open [lib/site-data.ts](./lib/site-data.ts) — every headline, bio, project,
stat, and social link lives there as a typed constant. Change values,
save, and Netlify will rebuild automatically on push.

## Design tokens

[styles/globals.css](./styles/globals.css) defines the design DNA — palette,
typography, spacing rhythm. Adjust the CSS variables there to reshape the
whole site.

## Deploying

This repo is wired to Netlify. Pushes to `main` trigger an automatic
`next build` + deploy. Live URL lives in the Netlify dashboard.
